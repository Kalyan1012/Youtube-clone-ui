# 📺 YouTube Clone - Responsive UI

A high-fidelity recreation of the YouTube homepage interface. This project focuses on modern CSS layouts, including **Flexbox** and **CSS Grid**, ensuring a pixel-perfect and responsive user experience.

## ✨ Key Features
* **Fully Responsive:** Adapts from mobile to desktop screen sizes.
* **Side Navigation:** Collapsible sidebar menu architecture.
* **Dynamic Grid:** Video thumbnail grid built with CSS Grid for scalability.
* **Search Architecture:** Header layout utilizing Flexbox for alignment.

## 🛠️ Technologies Used
* **HTML5:** Semantic structuring for accessibility.
* **CSS3:** Custom properties (variables), Flexbox, and Grid.
* **Iconography:** Using font-based or SVG icons.
